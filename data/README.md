# Local data (not in git)

Place large files here; `*.tif`, `*.nc`, and typical rasters are **gitignored**.

Suggested layout for the Palisades pilot:

```
data/palisades/planet/   # PlanetScope SR GeoTIFF (+ optional metadata JSON sidecar)
data/palisades/tempo/    # TEMPO L2 NetCDF and warped GeoTIFF outputs
```

See **PROJECT.md** (“Locked data”) for file naming.
